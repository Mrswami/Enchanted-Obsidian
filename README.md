# 🌌 Enchanted Obsidian
### // The Sovereign Intelligence Node

**Enchanted Obsidian** is a high-velocity, local-first Markdown engine designed for the **Sovereign Intelligence Protocol**. It bridges the difficult gap between **Autonomous Ease** (AI-driven organization) and **Total User Sovereignty** (The raw Markdown vault as the ground truth).

We maintain a perfect, yet challenging balance: The UI starts at the **Macro Level (The Command Hub)** to shield you from cognitive load and focus your attention on project sectors. As you **Deep-Dive** into a note, the interface peels back to the **Micro Level (Raw Prose)**—giving you absolute, unhindered control over your ideas.

---

## 🏛️ The Sovereign Intelligence Protocol
*   **The Focused Command Hub**: A high-fidelity, ergonomic dashboard for your Project Sectors (Enchanted, ChessPulse, PixelNinja, etc.).
*   **Deep-Dive Editability**: Direct access to your raw `.md` files. We maintain the "True Base" in your Obsidian vault at all times.
*   **Autonomous Ease (HITL)**: Proactive AI nudges for merging duplicates and splitting large notes—always present as suggestions, never executed without your command.
*   **The Ingestion Bridge**: A dedicated **Enchanted Brain** Discord bot that serves as your mobile-to-desktop bridge, manifesting ideas into your vault in real-time.

## 🍱 Tactical Layout (Keep-Style)
The UI is built for speed, featuring a **High-Density Masonry Grid** that lets you scan dozens of research blocks at once. Inspired by Google Keep, but hardened with a Carbon Nebula aesthetic.

## 🛠️ Infrastructure & Forge
*   **Rust Engine**: Powered by **Tauri 2.0** for native performance on Windows, Android, and iOS.
*   **Enchanted Forge**: Automated CI/CD pipeline in GitHub Actions that builds and distributes Android APKs.
*   **Mission Control (ETH Dashboard)**: A high-fidelity Azure-native dashboard for real-time telemetry, study sync, and shift management.
